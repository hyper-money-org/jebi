jebi icon redesign

A terminal prompt that happens to form a cube.

The current mark reads cube-first in the Dock. These six monochrome concepts invert the hierarchy so the first read is terminal, the second is developer tool, and the cube becomes the supporting idea.

Recommendation: 06 Dock Bold Hybrid

Concept 06 is the strongest fit for a modern AI-native terminal application because it solves the current hierarchy problem most aggressively: from distance, it reads as a large terminal prompt first, then as a block cursor, then as a cube.

Compared with the current icon, it avoids making the isometric cube the dominant silhouette. The cube is compressed into the cursor area, so the Dock read becomes terminal-first without abandoning the geometric cube idea that makes jebi feel distinct from a generic terminal emulator.
